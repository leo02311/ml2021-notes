# 第25堂課：GAN 的魔幻應用 - 非監督學習

## 緣起：GAN 在非監督學習的應用

本堂課將探討生成對抗網路（GAN）的一項「魔幻」應用：如何將其用於非監督學習（Unsupervised Learning）。這與我們之前討論的監督學習有所不同，因為非監督學習的資料不包含成對的輸入與輸出。

### 非監督學習的挑戰：不成對資料

在實際應用中，我們經常遇到擁有大量輸入 `X` 和大量輸出 `Y` 的情況，但 `X` 和 `Y` 之間卻沒有明確的成對關係。這類資料被稱為「未標記資料（unlabeled data）」或「不成對資料（unpaired data）」。

### 傳統監督學習方法的限制

過往的半監督學習（Semi-supervised Learning）方法，如作業三的偽標籤（pseudo labeling）或作業五的反向翻譯（back translation），雖然能利用部分未標記資料，但它們**仍需要一些成對資料作為基礎**來訓練初始模型或翻譯模型。當完全沒有任何成對資料時，這些方法便無法適用。

### 應用案例：圖片風格轉換

假設我們想訓練一個深度學習網路，將「領域 X」的圖片轉換為「領域 Y」的圖片。
*   **領域 X**：真人頭像
*   **領域 Y**：動畫角色頭像

這種轉換被稱為「圖片風格轉換（Image Style Conversion）」。然而，要取得成對的「真人 Yui Aragaki 照片」與「動漫版 Yui Aragaki 肖像」極為昂貴，甚至不可能。在這種完全沒有成對資料的情況下，我們如何訓練網路，讓輸入 `X` 產生 `Y` 呢？這正是 GAN 能夠發揮作用的場景。

## 初步嘗試：基於 GAN 的不成對影像轉換

### 核心概念調整

回顧過去的非條件式生成（Unconditional Generation）GAN，其生成器的輸入通常是高斯分佈的隨機向量，輸出是複雜的資料分佈（如真實圖片）。現在，我們可以對這個概念進行微調：

*   **生成器輸入**：來自「領域 X」的圖片分佈。
*   **生成器輸出**：希望其能產生「領域 Y」的圖片分佈。

要實現這一點，我們只需將原始 GAN 中「從高斯分佈取樣」的步驟，改為「從領域 X 的圖片中取樣」。例如，從大量真人臉部照片中隨機選取一張作為生成器的輸入。

而判別器的職責則是判斷生成器輸出的圖片是否屬於「領域 Y」。它會被訓練去辨識真正的領域 Y 圖片（給予高分）和非領域 Y 圖片（給予低分，如生成器輸出的圖片）。

### 單純應用 GAN 的局限性

然而，僅僅這樣修改，使用單一生成器和判別器是**不夠的**。

**問題點**：判別器只要求生成器輸出看起來像「領域 Y」的圖片。生成器可能會學會產生漂亮的動畫角色圖片，但這些圖片與原始的輸入 `X` **毫無關聯**。它可能將輸入圖片視為噪音，無論輸入什麼，只要輸出是動漫角色，判別器就認為它做得很好。這並不是我們想要的「風格轉換」，而更像是「無條件生成領域 Y 圖片」。

這個問題與之前在條件式 GAN 中遇到的挑戰類似：如果判別器只關注輸出 `Y`，它可能會忽略生成器的輸入 `X`。然而，由於是非監督學習，我們沒有成對資料來直接告訴判別器 `X` 和 `Y` 的正確組合。

## 解決方案：CycleGAN

為了解決生成器忽略輸入的問題，我們引入了 **CycleGAN**。

### CycleGAN 的核心思想

CycleGAN 的核心在於訓練**兩組生成器**：
1.  **G_XY**：將領域 X 的圖片轉換為領域 Y 的圖片。
2.  **G_YX**：將領域 Y 的圖片還原為領域 X 的圖片。

### 週期一致性損失 (Cycle Consistency Loss)

CycleGAN 引入了一個額外的訓練目標：**週期一致性（Cycle Consistency）**。
*   對於一張輸入圖片 `X`，首先通過 `G_XY` 將其轉換為領域 `Y` 的圖片 `Y'`。
*   然後，再通過 `G_YX` 將 `Y'` 還原回領域 `X` 的圖片 `X''`。
*   我們要求 `X''` 應該與原始輸入圖片 `X` **盡可能相似**。

這種「X -> Y -> X''」的循環轉換，其輸入與最終輸出的相似度作為損失函數，稱為週期一致性損失。它迫使 `G_XY` 的輸出不能完全脫離輸入 `X`，否則 `G_YX` 將無法成功地將其還原。

### CycleGAN 的運作機制

1.  **生成器 G_XY (X -> Y)**：將真人頭像 (X) 轉換為動畫角色頭像 (Y)。
2.  **生成器 G_YX (Y -> X)**：將動畫角色頭像 (Y) 轉換回真人頭像 (X)。
3.  **判別器 D_Y**：判斷 `G_XY` 生成的 `Y'` 是否像真實的 Y 領域圖片。
4.  **週期一致性**：確保 `X -> G_XY(X) -> G_YX(G_XY(X))` 得到的 `X''` 與 `X` 相似。

通過週期一致性，`G_XY` 被迫生成與輸入 `X` 有某種關聯的 `Y'`。例如，如果輸入是一個戴眼鏡的宅男，`G_XY` 至少會輸出一個戴眼鏡的動漫男孩（如新八），這樣 `G_YX` 才有機會將其還原為原來的宅男。

### 理論與實務上的考量

*   **理論局限**：週期一致性並非百分之百保證輸入 `X` 和輸出 `Y'` 之間是我們想要的「風格轉換」關係。例如，`G_XY` 可能學會將眼鏡擦掉變成痣，而 `G_YX` 則學會將痣變回眼鏡。甚至 `G_XY` 只是簡單地翻轉圖片，`G_YX` 再翻轉回來。這些都能滿足週期一致性，但不是理想的風格轉換。
*   **實務觀察**：雖然理論上存在這些風險，但在實際應用中，這種情況很少發生。網路通常「很懶」，傾向於學習簡單的、直接的轉換，而非複雜且奇怪的映射。所以，在大多數情況下，CycleGAN 的輸入和輸出圖片在內容上是相似的，只在風格上有所不同。

### 雙向 CycleGAN (Bidirectional CycleGAN)

CycleGAN 可以是雙向的。除了從 `X -> Y -> X'` 的週期外，還可以同時訓練另一個方向的週期：`Y -> X -> Y'`。
*   **G_YX**：將領域 Y 的圖片轉換為領域 X。
*   **G_XY**：將領域 X 的圖片還原為領域 Y。
*   **判別器 D_X**：判斷 `G_YX` 生成的 `X'` 是否像真實的 X 領域圖片。
*   同樣應用週期一致性：`Y -> G_YX(Y) -> G_XY(G_YX(Y))` 得到的 `Y''` 與 `Y` 相似。

這種雙向訓練進一步增強了模型在兩個領域間轉換的魯棒性。

## 相關與進階模型

### DiscoGAN 與 DualGAN

與 CycleGAN 幾乎在同一時間（2017年3月至4月），由不同團隊獨立提出了類似概念的模型，如 **DiscoGAN** 和 **DualGAN**。它們的核心思想和架構與 CycleGAN 非常相似，體現了同時期的研究趨勢。

### StarGAN

**StarGAN** 是 CycleGAN 的進階版本，它能夠在**多個風格**之間進行轉換，而不僅限於兩個風格。這擴展了風格轉換的應用範圍。

### 實際應用案例：真人轉動漫

目前已有一些基於 GAN 的技術能將真人照片轉換為動畫風格。例如，某韓國團隊的網站便提供了此功能。雖然他們可能未使用 CycleGAN，而是更先進的 GAN 技術，但成果展示了這類應用的潛力。

*   **成功案例**：新垣結衣的照片能成功轉換，眼睛被放大，呈現動漫特徵。
*   **失敗案例**：美國前總統的照片轉換後，可能出現一隻眼睛大一隻眼睛小的失誤，顯示技術仍有局限性。

## 其他應用領域

CycleGAN 的思想不僅限於圖片風格轉換，還可以推廣到其他多種類型的資料轉換任務。

### 文字風格轉換 (Text Style Transfer)

*   **目標**：將負面情緒的句子轉換為正面情緒的句子。
*   **資料**：從社群媒體（如 PTT）爬取大量不成對的正面和負面評論（例如根據按讚/噓文）。
*   **方法**：
    *   訓練兩個生成器：`G_Neg2Pos` 和 `G_Pos2Neg`。
    *   訓練判別器 `D_Pos` 和 `D_Neg` 來判斷句子風格。
    *   應用週期一致性：`Neg -> G_Neg2Pos(Neg) -> G_Pos2Neg(G_Neg2Pos(Neg))` 應與原始 `Neg` 相似。
*   **挑戰**：
    *   如何衡量兩個句子之間的相似度？（相較於圖片向量距離更複雜）
    *   如何讓判別器處理文本輸出？（通常需要結合強化學習）
*   **範例**：
    *   輸入：「肚子痛、睡不飽、心情差」 → 輸出：「生日快樂、睡得好、心情很棒」
    *   輸入：「我甚至想去上班了，真可悲」 → 輸出：「我甚至想去睡覺了，真酷」
    *   輸入：「爛透了，吃烤肉還遇到露鳥俠」 → 輸出：「棒呆了，吃烤肉還遇到帥哥」 (創造了「帥哥」一詞來對應「露鳥俠」)
*   **有趣現象**：有時會產生「胡言亂語」的結果，如「My stomach is hurting bad」轉換成「My happy birthday is going good」。機器將所有腹部症狀都轉換成「生日快樂」，顯示其學習到某些模式，但缺乏語義理解。
*   **實用性**：教授幽默地表示這項技術目前「完全沒用」，但如果對老闆懷恨在心，可以將批評言論轉換成讚美，活出更好的人生。

### 文件摘要 (Document Summarization)

*   **目標**：將長文件摘要成短文。
*   **資料**：大量不成對的長文件和短摘要。
*   **方法**：利用 CycleGAN 思想，訓練模型從長文件生成摘要，並從摘要重建長文件，達成風格轉換。

### 非監督式機器翻譯 (Unsupervised Machine Translation)

*   **目標**：在沒有任何成對翻譯資料的情況下，實現不同語言（如中英）之間的翻譯。
*   **資料**：僅有大量英文句子和大量中文句子，但它們沒有任何對應關係。
*   **方法**：直接採用 CycleGAN 方法，將英文轉換為中文，再將中文轉換回英文，並確保週期一致性。

### 非監督式語音辨識 (Unsupervised Automatic Speech Recognition - ASR)

*   **目標**：在沒有語音與文字轉錄成對資料的情況下，實現語音辨識。
*   **資料**：只有大量的語音訊號，和從網路爬取的大量文本資料，但兩者之間沒有對應關係。
*   **方法**：李宏毅教授實驗室的成員曾率先進行此類研究，應用 CycleGAN 方法，嘗試將語音訊號轉換為文本，並檢驗其效果。

## 總結

本堂課介紹了 GAN 在非監督學習領域的「魔幻」應用，特別是 CycleGAN 如何通過週期一致性解決了不成對資料下的風格轉換問題。從圖片到文本，再到更複雜的語音轉換，CycleGAN 的思想為處理無成對資料的任務開闢了新的可能性。GAN 部分的課程至此告一段落。
